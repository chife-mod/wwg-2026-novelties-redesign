# Top brands

**Problem.** Оригинальная (A) плитка — плоский список с маленькими лого (24px), серым kicker'ом «MAISONS» и префиксом «№» перед рангом. Логотипы теряются, заголовок тяжёлый, иерархия не читается с первого взгляда.

**Tried.**
- A (Before · original) — baseline, избыточный chrome.
- B (Watch360 native) — сохранил родные пропорции дизайн-системы, но всё ещё flat.
- C (rank № 01) — вернул крупные цифры, но «№» повторяет себя, а kicker остался.
- D (gold title, bare numbers, bigger logos) — ✓ **selected**. Заголовок «Top brands» gold #A98155 работает сам eyebrow'ом, убрал «№», логотипы 32px, row min-h 32, gap 8px.
- D₁ (large deltas with % suffix) — stress-test для 3/4-значных YoY процентов, живёт только в sandbox.

**Decision.** Gold-заголовок переносит акцент с kicker'а на сам title (одна сущность вместо двух). Голые номера и крупные лого дают brand-first-чтение. Общий CSS grid на весь список через `<li class="contents">` фиксит правую колонку count/delta между строками — значения больше не скачут.

**Notes.** D₁ хранит gотовый паттерн, если придут реальные YoY % с 3-4 цифрами: столбец дельт не ломается, `min-w-[42px]` + `justify-self-end` + общий grid-трек держат все строки выровненными.
