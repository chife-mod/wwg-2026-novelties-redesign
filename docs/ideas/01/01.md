# Идея №1 — Окно в выставку + слайдер топ-моделей

**Дата:** 2026-04-20
**Автор:** Олег (pm/дизайнер)
**Статус:** захвачена, ждёт реализации в V1

## Суть

Левая часть страницы — **визуальная зона**, окно в реальный мир выставки. Вместо очередных бар-чартов пользователь сразу видит *саму выставку*:

- зацикленное видео с выставочных залов (обои / motion background);
- или лайв-трансляция с WWG, если доступна;
- или хотя бы кинематографичное статичное фото-интро, переходящее во встроенный плеер.

Справа от визуала — **слайдер топ-10 моделей**. Не плитка с бар-чартом, а карточки, которые «вылетают» по очереди. На каждом слайде:

- большое фото часов;
- лого бренда;
- название модели;
- reference number;
- цена;
- ключевые характеристики в виде тегов-чипов (*Blue dial · Golden case · 40 mm · Alligator strap*).

Смысл: **перенести пользователя из скучной таблицы в реальный мир выставки**. Ощущение присутствия. Одним экраном мы показываем и *вайб выставки*, и *конкретные модели, которые её определяют*.

## Что это меняет в V1

Эта идея **заменяет Tier 1** из текущего дизайн-дока (статичные карточки Top Brands + Top Collections) на связку:

- **левая колонка (≈55–60% ширины):** окно-видеовизуал выставки;
- **правая колонка (≈40–45%):** слайдер топ-10 моделей с автосменой и навигацией (стрелки / индикаторы / пауза).

Минус одна «плитка с бар-чартами» из старой страницы. Остальная иерархия остаётся: Hero-strip → **Fair Window + Slider** → Headline Insights → Price / Dial / Diameter → By the Numbers → Footer.

## Почему это работает

- **Вау с первого экрана.** Ни у одной страницы Watch360 такого сейчас нет — это и пресс-материал, и портфолио-вещь.
- **Эмоция + факт в одном блоке.** Левая часть — настроение, правая — конкретика (модель, цена, характеристики).
- **Реюз паттерна слайдера.** Слайдер у нас уже когда-то делался — форма знакомая, остаётся адаптировать контент.
- **Минус один скучный фасет.** Коллекции/модели теперь подаются визуально — а не ещё одним ranked-list.

## Что нужно решить перед реализацией

1. **Источник визуала на прототипе:** зацикленный video-placeholder (CC-видео выставочного зала), motion-обои (CSS parallax поверх фото), или статичное фото с Ken-Burns эффектом? На первом проходе проще всего статичное фото + лёгкий scale/pan.
2. **Где взять фотографии часов для слайдера:** на прототипе — плейсхолдер-композиции (круглая «часовая» форма + монограмма бренда + лейаут спецификации), как договаривались. Позже — настоящие рендеры от брендов.
3. **Правила автопрокрутки:** интервал 5–7 сек, пауза при hover, индикаторы снизу, клавиши ← →.
4. **Оверлей с UI поверх видео:** бейдж «LIVE · Geneva · Hall 1», счётчик «Day 2 of 5», микроконтролы.
